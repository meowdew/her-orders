# Her Orders

Her Orders 是一款为女朋友专属设计的微信小程序。它把日常生活里的想法、心愿、待办和小请求整理成清晰的“订单”，让男朋友可以更轻松地接收、确认和完成她的想法。

这个项目的目标不是做一个复杂的商业订单系统，而是打造一个轻量、贴心、好用的亲密关系协作工具：她可以随时写下想要的事，他可以看到状态、及时回应，并把每一次小心愿都认真对待。

## Features

- 想法和心愿收集
- 订单式状态管理
- 今日待办概览
- 轻量、私密、专属的微信小程序体验

## Tech Stack

- 微信小程序原生开发
- TypeScript
- npm package support

## Development

Install dependencies:

```bash
npm install
```

Run checks:

```bash
npm run check
```

Open this repository with WeChat DevTools, then compile or preview the mini program.

## Scope

This repository focuses on the WeChat Mini Program. Native iOS development will live in a separate Swift repository.
